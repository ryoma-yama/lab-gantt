> LabGantt visualizes GitLab issues as Gantt charts

### ✨ [Live Demo](https://ryoma-yama.github.io/lab-gantt/)

## Prerequisites

- node >=20.0.0
- corepack >=0.28.2

Corepack not set up? Check [pnpm setup guide](https://pnpm.io/installation#using-corepack).

## Install

```sh
pnpm i
```

## Usage

```sh
pnpm dev
```
